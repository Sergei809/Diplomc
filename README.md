# Проект автоматизации тестирования Hudson Store

Этот проект представляет собой автоматизированные тесты для веб-сайта Hudson Store с использованием Selenium WebDriver и Python.

## Структура проекта

```
Diplomc/
├── base/                    # Базовые классы
│   ├── __init__.py
│   └── base_class.py       # Базовый класс с общими методами
├── pages/                   # Page Object Model классы
│   ├── __init__.py
│   ├── login_page.py       # Страница авторизации
│   ├── first_page.py       # Главная страница с навигацией
│   ├── main_page.py        # Страница с товарами
│   └── client_information_page.py  # Страница информации о клиенте
├── tests/                   # Тесты
│   ├── __init__.py
│   └── test_buy_product.py # Тест покупки товара
├── screen/                  # Папка для скриншотов
├── utilites/               # Утилиты
├── requirements.txt        # Зависимости проекта
└── README.md              # Документация
```

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Установка Firefox WebDriver

Убедитесь, что у вас установлен Firefox браузер. Selenium автоматически управляет драйвером через WebDriver Manager.

### 3. Запуск тестов

```bash
# Запуск всех тестов
python -m pytest -s -v tests/

# Запуск конкретного теста
python -m pytest -s -v tests/test_buy_product.py

# Запуск теста с определенным порядком
python -m pytest -s -v tests/test_buy_product.py::test_buy_product_1
```

## Описание тестов

### test_buy_product_1
Тест проверяет полный процесс покупки товара:
1. Авторизация на сайте
2. Переход в категорию "Men"
3. Выбор товара
4. Добавление в корзину

## Особенности реализации

1. **Page Object Model** - каждый класс представляет отдельную страницу
2. **Явные ожидания** - использование WebDriverWait для надежности
3. **Базовый класс** - общие методы вынесены в базовый класс
4. **Скриншоты** - автоматическое создание скриншотов при ошибках
5. **Логирование** - подробные сообщения о выполняемых действиях

## Возможные проблемы

1. **Элементы не найдены** - сайт может изменить структуру, потребуется обновление селекторов
2. **Таймауты** - медленное соединение может потребовать увеличения времени ожидания
3. **Браузер** - убедитесь, что Firefox установлен и обновлен

## Разработка

Для добавления новых тестов:
1. Создайте новый класс страницы в папке `pages/`
2. Наследуйтесь от `Base` класса
3. Добавьте локаторы, геттеры, действия и методы
4. Создайте тест в папке `tests/`
5. Импортируйте необходимые классы страниц
